# Ứng dụng Machine Learning trong y tế: Dự đoán nguy cơ bệnh tim

Bản đã sửa:
- Sidebar có thể bấm chuyển mục
- Form nhập dữ liệu dùng number_input dễ điền
- Có 3 thuật toán: Hồi quy Logistic, Cây quyết định, Random Forest
- Có so sánh Accuracy và F1-score
- Có dữ liệu CSV, biểu đồ và đánh giá mô hình

## Chạy ứng dụng

```bash
streamlit run app.py
```
